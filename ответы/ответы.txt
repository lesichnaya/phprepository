1. Типизация в php -  динамическая слабая неявная. В динамической типизации все типы выясняются во время выполнения программы.
Неявные преобразования выполняются автоматически. Типы переменных не нужно указывать.
2. Типы данны: integer  - целове число, float - число с плавающей запятой, bool(логический) - true, false, string - строка
3.  В ассоциативном массиве вместо индекса в качестве ключа используется строка. При создании такого массива явным образом указывается ключ элемента. Далее идет оператор =>, а затем значение элемента
4. Операторы сравнения
	== равенства
	=== оператор тождественности (дополнительно сравнивает тип значение)
	!= значения не равны
	!== аналгично оператору тождественности
	>, <  больше, меньше
	>=, <=  больше или равно и меньше или равно
	<=> - возвращает 0, если оба значения равны, 1, если значение слева больше, –1, если значение слева меньше, чем значение справа
5. Бывают циклы: for, while, do while, foreach.
6. foreach используется для массивов. При каждой итерации текущему элементу массива присваивается значение, а указатель массива перемещается на одну позицию, в следующей итерации будет обрабатываться следующий элемент.
